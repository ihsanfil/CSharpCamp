# 👨‍💻 C# EĞİTİM KAMPI

Bu proje, Murat Yücedağ'ın C# Eğitim Kampı online kursuna dayanarak hazırlanmıştır. C# dilini öğrenmek ve uygulamalı pratik yapmak isteyenler için oluşturulan bu repo, temel, orta ve ileri seviyedeki konuları içeren dersleri kapsamaktadır. 🎯

## 🚀 C# 101 - Temel Programlama
+ Giriş, Merhaba Dünya
+ Değişkenler: String, Int, Double, Char
+ Karar Yapıları: If, Else, Switch
+ Döngüler: For, While
+ Döngüler Pratik Çalışma
+ Diziler
+ Foreach Döngüsü
+ Metotlar

## 🗄 C# 201 - SQL ve Veri Yönetimi
+ SQL Veri Tabanı ve Tablo Oluşturma & C# Kısmına Bağlama
+ C# ile SQL Listeleme Ekleme Güncelleme ve Silme İşlemleri

## 🔥 C# 301 - OOP ve Entity Framework, N Katmanlı Mimari Entity Layer
+ OOP Modülü: C# ile Nesne Tabanlı Programlama
+ OOP Modülü: Data Access Katmanı
+ OOP Modülü: Migration Kavramı
+ ORM Yapısı: Entity Framework Giriş
+ Entity Framework Metotları ile Veri İşlemleri
+ Entity Framework: Tur Projesi Uygulaması
+ Entity Framework Metotları ve Query Kullanımı
+ EntityState Komutları, Generic Repository
+ Business Katmanı ve Logic Yazımı
+ Dependency Injection

## ⚡ C# 401-501 - İleri Seviye C# ve Veri İşlemleri
+ Entity’ye Özgü Metot Yazmak
